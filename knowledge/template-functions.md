# Chezmoi Template Functions Reference

## Built-in Variables

### chezmoi Variables

```
{{ .chezmoi.os }}           # OS: darwin, linux, windows
{{ .chezmoi.arch }}         # Architecture: amd64, arm64, 386
{{ .chezmoi.hostname }}     # Hostname
{{ .chezmoi.username }}     # Username
{{ .chezmoi.homeDir }}      # Home directory path
{{ .chezmoi.configFile }}   # Config file path
{{ .chezmoi.sourceDir }}    # Source directory path
{{ .chezmoi.version }}      # chezmoi version
```

### Custom Data

Define in `chezmoi.toml`:

```toml
[data]
    name = "Your Name"
    email = "you@example.com"
    company = "ACME"
```

Use in templates:

```
{{ .name }}
{{ .email }}
{{ .company }}
```

## Conditionals

### Operating System

```
{{ if eq .chezmoi.os "darwin" }}
# macOS config
{{ else if eq .chezmoi.os "linux" }}
# Linux config
{{ else }}
# Other systems
{{ end }}
```

### Hostname

```
{{ if eq .chezmoi.hostname "work-laptop" }}
# Work machine config
{{ else }}
# Personal machine config
{{ end }}
```

### File Existence

```
{{ if stat (joinPath .chezmoi.homeDir ".work") }}
# Work environment
{{ end }}
```

## Common Functions

### String Processing

```
{{ .name | upper }}         # Uppercase
{{ .name | lower }}         # Lowercase
{{ .name | title }}         # Title case
{{ .name | trim }}          # Trim whitespace
{{ .email | replace "@" " at " }}  # Replace
```

### Path Processing

```
{{ joinPath .chezmoi.homeDir ".config" "app" }}
{{ .chezmoi.homeDir | trimPrefix "/Users/" }}
```

### Output Command Results

```
{{ output "date" "+%Y-%m-%d" }}
{{ output "hostname" | trim }}
```

### Get Password from Keychain

```
{{ onepasswordRead "op://vault/item/field" }}
{{ bitwardenFields "item" | jq ".field" }}
{{ lastpassRaw "item" }}
```

## Advanced Usage

### Loops

```
{{ range .packages }}
install {{ . }}
{{ end }}
```

With configuration:

```toml
[data.packages]
    = ["git", "vim", "zsh"]
```

### Comment Generation

```
# Generated by chezmoi on {{ now | date "2006-01-02" }}
# Do not edit directly
```

### Include Other Templates

```
{{ template "common" . }}
```

## Complete Examples

### Git Config Template

```
# dot_gitconfig.tmpl
[user]
    name = {{ .name }}
    email = {{ .email }}
{{ if eq .chezmoi.hostname "work-laptop" }}
    signingkey = {{ .work_gpg_key }}
{{ else }}
    signingkey = {{ .personal_gpg_key }}
{{ end }}

[core]
    editor = {{ .editor | default "vim" }}
{{ if eq .chezmoi.os "windows" }}
    autocrlf = true
{{ else }}
    autocrlf = input
{{ end }}

[includeIf "gitdir:~/work/"]
    path = ~/.gitconfig-work
```

### SSH Config Template

```
# private_dot_ssh/config.tmpl
Host github.com
    HostName github.com
    User git
    IdentityFile ~/.ssh/id_ed25519
{{ if eq .chezmoi.hostname "work-laptop" }}

Host work-git
    HostName git.company.com
    User git
    IdentityFile ~/.ssh/id_work
{{ end }}
```

### Zsh Config Template

```
# dot_zshrc.tmpl
# OS: {{ .chezmoi.os }}
# Host: {{ .chezmoi.hostname }}

export EDITOR={{ .editor | default "vim" }}
export PATH="$HOME/bin:$PATH"

{{ if eq .chezmoi.os "darwin" }}
# macOS specific
export PATH="/opt/homebrew/bin:$PATH"
{{ else if eq .chezmoi.os "linux" }}
# Linux specific
export PATH="$HOME/.local/bin:$PATH"
{{ end }}

{{ if stat (joinPath .chezmoi.homeDir ".work") }}
# Work specific settings
source ~/.zshrc-work
{{ end }}
```
